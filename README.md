# Sales Performance Analysis Dashboard
Project Overview
## This Power BI project provides a comprehensive analysis of sales 
performance, aiming to identify key trends, top-performing 
products, and regional growth opportunities. The dashboard 
transforms raw sales data into actionable business insights.
Key Features
## * *Dynamic Dashboard:* Interactive visuals allowing users to 
filter by date, region, and product category.
* *Sales Trends:* Time-series analysis to identify seasonal peaks 
and growth patterns.
* *KPI Tracking:* Real-time monitoring of Total Revenue, Profit 
Margins, and Order Volumes.
* *Product Performance:* Deep dive into top-selling items and 
underperforming categories.
Tools & Technologies
## * *Power BI Desktop:* For report authoring and dashboard design.
* *Power Query:* Used for data cleaning, transformation, and ETL 
processes.
* *DAX (Data Analysis Expressions):* Created complex measures for 
calculating YoY growth and profit ratios.
* *GitHub:* For version control and project documentation.
##Project Structure 
* /Report: Contains the Power BI report layout and visuals.
* /SemanticModel: Contains the data model, relationships, and DAX 
measures.
* sales project(power pi).pbip: The main project file.
